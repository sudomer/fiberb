
# Fiber v2 Boilerplate

Hi, This repo was created to avoid some basic mongodb queries and retyping API endpoints

## Used libraries

- Fiber v2

- MongoDB with official mongodb package

- Validation

- Zap ( for logging )

## Installation & Usage

```bash 
  git clone https://github.com/sudomer/fiberb.git
  cd fiberb
  go mod tidy
  go run .
  
```
